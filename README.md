## 💡 About This App

Agentpet acts as a desktop companion for your coding workflow. It monitors your AI coding agents, such as Claude Code, Codex, and the Gemini CLI, while they complete tasks. The pet sits in your menu bar and provides updates on your system status. This gives you a clear view of your development progress without needing to switch windows or monitor terminal text.

## 📋 System Requirements

To run agentpet, ensure your computer meets these conditions:

- Operating System: Windows 10 or Windows 11 (64-bit).
- Processor: Any modern dual-core processor or better.
- Memory: At least 4GB of RAM.
- Storage: 100MB of free disk space.
- Internet Connection: Active connection to download and receive updates.

Agentpet includes several tools to help you stay focused during coding sessions:

- Status Notifications: Receive simple alerts if your coding tool encounters an error or finishes a complex task.

## 🛠 Troubleshooting Common Issues

If you experience problems, consult these common solutions:

- The app does not launch: Ensure you have the latest version installed. Try reinstalling if the icon does not appear in your tray.
- Connection errors: Check your firewall settings. The app requires access to monitor local command-line processes.
- High memory usage: Restart the app. If the issue persists, check that you are running the most recent version from the GitHub releases page.
- Missing AI tool detection: Verify that your AI coding CLI is installed correctly and is currently active in your terminal. Agentpet listens for signals from these specific software tools.

## 🛡 Security and Privacy

Agentpet interacts only with the status signals from your local AI coding tools. The application does not read your private source code or sensitive data. It only tracks the activity state of the specific processes you select. Your data remains on your local machine.
